# EcoMark

EcoMark is a lightweight application focused on promoting eco-friendly choices through a simple and accessible digital experience.

## Features

* User-friendly interface
* Eco-awareness tracking (or scoring)
* Lightweight and fast performance

## Tech Stack

This project is built using:

* **JavaScript / TypeScript** – Core application logic
* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **Markdown** – Documentation

## Project Structure

```
/src        → Main application code  
/public     → Static assets  
README.md   → Project documentation  
```

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/mitch-techie/EcoMark.git
   ```
2. Navigate into the folder:

   ```bash
   cd EcoMark
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Run the project:

   ```bash
   npm start
   ```
